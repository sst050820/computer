# ABE 策略 → 业务映射

## 条件类型

| 业务条件 | 字段名 | ABE 策略片段 | 前端选项 |
|---------|--------|-------------|---------|
| 产地要求 | Location | `Location=福建` | 福建/山东/浙江/云南/安徽/四川/广东 |
| 加工能力 | Capability | `Capability=制茶` | 制茶/果蔬加工/糕点制作/酿造/干货加工/冷冻加工 |
| 品质认证 | Quality | `Quality=有机` | 有机/绿色/地理标志/无公害 |
| 等级要求 | Grade | `Grade=3` | 1/2/3/4/5 |
| 有机认证 | Organic | `Organic=是` | 是/否 |

## 策略表达式格式

```
(属性1=值1, 属性2=值2; N)
```
- N = 条件数量（LSSS 访问矩阵阈值）
- 所有条件必须同时满足（AND 逻辑）

## 策略示例

| 业务需求 | ABE 策略 | 说明 |
|---------|---------|------|
| 只要福建工厂 | `(Location=福建; 1)` | 单条件 |
| 福建 + 制茶能力 | `(Location=福建, Capability=制茶; 2)` | 双条件 AND |
| 福建 + 制茶 + 有机 + 3级 | `(Location=福建, Capability=制茶, Quality=有机, Grade=3; 4)` | 四条件 AND |

## 资质 → 属性映射

| 资质（业务层） | 属性（ABE 层） | 颁发方 |
|---------------|---------------|--------|
| 产地=福建 | Location=福建 | 工商认证中心 |
| 能力=制茶 | Capability=制茶 | 工商认证中心 |
| 品质=有机 | Quality=有机 | 有机认证协会 |
| 等级=3 | Grade=3 | 工商认证中心 |

## 完整 ABE 流程

```
┌─────────────────────────────────────────────────────────┐
│ 1. 消费者设定条件                                         │
│    ConditionSelector → conditions = {Location:福建, Capability:制茶} │
├─────────────────────────────────────────────────────────┤
│ 2. 前端生成策略预览                                       │
│    ConditionSelector.toPolicy() → "(Location=福建, Capability=制茶; 2)" │
├─────────────────────────────────────────────────────────┤
│ 3. Go 后端转换                                           │
│    service.ConditionToPolicy() → ABE 策略字符串           │
│    service.EncryptWithABE() → Java CryptoServer          │
├─────────────────────────────────────────────────────────┤
│ 4. Java ABE 加密                                         │
│    MAFASACAR.Enc(l, k) → LSSS 矩阵 + 密文               │
│    返回 sessionID + ciphertext                           │
├─────────────────────────────────────────────────────────┤
│ 5. 存储                                                  │
│    密文存入内存 / Fabric CreateFruit                     │
├─────────────────────────────────────────────────────────┤
│ 6. 商家查询                                              │
│    Go 获取商家资质列表 → 逐一比对需求条件                  │
│    匹配 → 返回完整描述                                    │
│    不匹配 → "🔒 不具备查看资质"                            │
└─────────────────────────────────────────────────────────┘
```

## 策略转用户可读文本

```go
// Go: service.PolicyToDisplay()
"(Location=福建, Capability=制茶; 2)"
  → "产地要求：福建 | 加工能力：制茶"

// 前端直接展示后者
```

## MAFASAC-AR 算法参数

| 参数 | 说明 |
|------|------|
| 群 | 复合阶双线性群 (Type A1, 256-bit) |
| LSSS 矩阵 | Vandermonde 矩阵 |
| 授权方数量 | 等于全局属性数量 |
| 属性撤销 | 支持 (SysUpd/KeyUpd/CTUpd) |
| 访问结构 | (l, k) — l 行 k 列矩阵 |
