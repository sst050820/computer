# 隐农链 — 五角色功能设计

## 角色总览

| 角色 | 图标 | ABE 对应 | 核心职责 |
|------|------|---------|----------|
| 消费者 | 🛒 | DataOwner | 浏览商品、发布定制需求（设定访问策略） |
| 商家/工厂 | 🏭 | DataUser | 商品管理、查看匹配需求（持有属性密钥） |
| 资质审核方 | 🏛️ | AttributeAuthority | 审核商家资质、颁发/收回属性密钥 |
| 平台管理员 | ⚙️ | 超级用户 | 全局管理、纠纷仲裁、规则更新 |
| 监管方 | 🔍 | DataUser (监管密钥) | 产品追溯审查、应急解密 |

## 一、消费者 (consumer)

**账号**: `zhangguonong` / `123456`

| 页面 | 功能说明 |
|------|---------|
| 发现好物 | 搜索/筛选/浏览农产品，关键词+品类+产地多维筛选 |
| 商品详情 | 查看商品信息、认证标签、溯源标识 |
| **私人定制** ⭐ | 设定匹配条件（产地/能力/品质）→ 自动生成 ABE 策略 → 定向发布 |
| 定制广场 | 浏览他人公开发布的定制需求摘要 |
| 我的需求 | 管理已发布的定制、查看工厂响应和报价 |
| 购物车 | 待结算商品 |
| 消息中心 | 工厂响应通知、订单消息、系统通知 |
| 个人中心 | 查看/编辑个人信息 |

### 私人定制流程
1. 点击「私人定制」→ 填写需求标题、描述、预算
2. 在条件选择器中设定：产地要求 / 加工能力 / 品质认证 / 等级要求
3. 系统实时预览生成的 ABE 策略表达式
4. 点击「定向发布」→ 需求加密 → 满足条件的工厂才可见

|------|---------|

| 商品管理 | 发布/编辑商品、查看商品列表 |
| 发布商品 | 填写商品信息、上传检测报告、设置信息可见范围 |
| 产品档案 | 为商品添加追溯节点（种植/加工/质检/运输/到店） |
| **需求市场** ⭐ | 系统根据资质自动匹配的定制需求、匹配/不匹配分类展示 |
| 需求详情 | 查看完整需求内容（仅匹配时可见） |
| 我的响应 | 已响应的定制需求及状态跟踪 |
| **我的资质** ⭐ | 查看/申请/续期资质、资质标签可视化 |
| 通行凭证 | 凭证状态查看与更新 |
| 订单管理 | 订单处理、状态跟踪 |
| 店铺设置 | 编辑店铺信息 |

### 需求市场 ABE 匹配
- 完全匹配：绿色高亮，显示完整需求详情，可接单
- 部分匹配：黄色标记，显示条件对比（要求 vs 现有），不可见详情
- 不匹配需求不出现在列表中

## 三、资质审核方 (certifier)

**账号**: `fujiangongshang` / `123456`（福建省工商认证中心）

| 页面 | 功能说明 |
|------|---------|
| 工作台 | 审核数据概览 |
| 审核管理 | 待审核资质列表、通过/驳回操作 |
| 审核详情 | 查看申请详情和证明材料 |
| 审核历史 | 已处理的审核记录 |
| 资质管理 | 已颁发资质列表、续期/收回操作 |
| 收回资质 | 填写收回原因，确认后资质立即失效 |
| 管辖范围配置 | 配置本机构管辖的资质类型 |
| 统计分析 | 月度趋势、类型分布 |
| 操作记录 | 资质授予/更新/收回日志 |
| 机构信息 | 机构基本信息管理 |

### ABE 对应
审核方对应 MAFASAC-AR 的 `AttributeAuthority`：
- 审批通过 → 颁发属性密钥 (KeyGen)
- 收回资质 → 撤销属性密钥 (SysUpd + KeyUpd)
